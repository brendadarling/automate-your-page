HTML Foundation

HTML is considered the foundation of a page, while CSS is the decoration and style of a page. JS is the interactive components.

Tree-like Structures (branching structures) of HTML. A tree-like structure is like the branches on a tree. Different branches with different styles. The structures appear as boxes. Boxes inside of boxes inside of more boxes.These boxes consist of an HTML,CSS, and DOM.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

It is very important that programmers avoid repetition for many reasons.

Avoiding Errors:When a programmer does not avoid repetition, they do the same thing over and over. Then when a programmer decides to make changes it tends to lead to errors. It is important that programmers make the same change to all relevent areas to avoid errors.

Consistency

Consistency is key to programming codes. When a programmer specifies the same style for similar codes. This makes it easier to make changes. For example applying the class code of "class".

Understanding CSS

CSS stands for Cascading Style Sheets. CSS allows programmers to have control of how their HTML docs will look. CSS can be thought of as search and replace tools. Cascading means there are rules that are applied to an element that is a direct match, as well as they are applied to the child elements, unless the child element has multiple overlapping rules that are defined to it then those rules take affect, this is call "Inheritance". The properties that can be inherited are color, font, letter-spaces, line height, list-style, text-align, text-indent, text-transform, white space, and word spacing.

Box Sizing and Positioning

As we learned in a previous lesson, HTML elements are boxes, each box has four components. Being as there are so many components to each box it can be difficult to size the box just right.
There are two techniques that are useful with sizing, one is to set the size in percentage as opposed to pixels, the other is to utilize the box sizing border box attribute.
Positioning: Div's are block elements, and by default they use the entire page. Adding rules of display flex in the CSS will override the default and allow the DIV's to appear next to each other. The "Box" is the margin, border, padding and content.

Computer

Computers are universal machines that we can program to essentially any computation that programmers program. Programmers need to be precise with their programs and codes. If a programmer makes a mistake, the computer does not correct the mistake. Remember computers are stupid.

Computer Program

A computer program is a list of instructions that tells a computer what to do. Programmers tell the computer what to do by using the programs. Programs are really the core of computer science, without programs a computer is "useless".

Python

Pyhton is a high level computer language that we can use to write programs. Python is called an Interpreter, it runs the programs, interprets them and then executes them. Python emphasizes code readability.

Computer Language

Computer Language is a formal constructed language designed to communicate intructions to a computer. It is used to create programs to control the behavior of a computer. There are reasons we do not use "Natural" language to program. It is amibiguous and verbose. Different people will interpret the same sentence many different ways. An example of Computer Language is the Backus-Naur Form. The purpose for this is to allow programmers to precisely describe the language in a way that is simple and consise.

Python Expression

A Python expression is a math expression of sorts. For example you can print 4+4 and run this, which will result in 8. A valid expression must have an open and a close. Python expression=expression-operator-expression. Expressions represent things like, numbers, strings or an instance of class. Any value is an expression. The open expression and the close expression. For example: print 11+ 11 is a valid expression, but print 11 + (without a number at the end) is not.

Lesson 6: Input-> Function-> Output

What is a function?

A function takes input, does something to that input, which then produces output. For example, a function named square can take a number as input and the output will be the input squared. Function can be a type of "procedure".

What is the difference between making and using a function?

Making functions are a line of code that begin with def. In the body of the function a code is input that tells what to do within the parameters of that input. For example SQUARE: following code could be the definition of a function called square:

def square(x):
answer = x * x
return
Now to use a function , we write the names of the function followed by the value(s) we want it to give in parentheses.
See example:
print square(4)
>>>16
How do functions help programmers avoid repetition?

Once programmers have created a function that function will never have to be created again.

What happens if a function does not have a return statement?

Since return tells Python what the function should produce as output. If a function is void of a return statement, then there will be no return output. Nothing will happen. See below:

Debugging

Debugging is a way to locate and correct errors.

Examine errors messages when programs crash.
The last line of Python Tracebacks will tell you what went wrong. Read the errors starting with the last line and read backwards, this will tell you where the errors is.

Work from example code.

Lesson 8: Structured Data:Lists and For Loops

Nested Lists

List data is a sequence of anything and they are much more powerful than strings.
Nested lists are elements that can be anything, and any number of elements. They show the versatility of lists. However it is not very useful as they do not have structure to what we put in them They are useful to the extent that we can put anything in them such as a list in a list.

For example:
kids = [['Victoria', 1983],['Cheryll', 1985],['John', 1988],(3)]
print kids
Mutation

Lists support mutation, I know it sounds scary. We can change the value of the list after we have created them. Mutation modifies the existing object within an element.

Aliasing

Aliasing can be thought of as two variable names with the same property. Two different ways to refer to the same object. Any changes made to either variable will change both variables, however if you change the element the other variable will not change.

List Operations

There are many built-in on lists.
Append, this method adds a new element to the end of a list. Append method mutates the list that it is invoked on, it does not create a new list.

stooges = ['Moe', 'Larry', 'Curly']
print stooges
['Moe', 'Larry', 'Curly']
stooges .append('Shemp')
print stooges
['Moe', 'Larry', 'Curly', 'Shemp']

Concatenation. The plus operator can be used with lists and is similar to how it is used to concatenate strings. It produces a new list, it does not mutate either of the input lists.

For example:
[0, 1] + [2, 3] -> [0,1,2,3]

Length. The len operator can be used to find the length of an object. The len operator works for many things other than lists, it works for any object that is a collection of things including strings. The output is the number of elements in its input.

For example:
len([0,1]) -> 2
len(['a', ['b', ['c','d']]]) -> 2
When you use a len on a string the output is the number of elements in the string.
For example:
len("Family") -> 6
